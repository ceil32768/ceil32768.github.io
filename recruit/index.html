關於計畫

遊戲總監 / 發起人 Ceil

我是 Ceil，Ceilusion 的創辦人兼遊戲總監。

我相信，遊戲是世界上最強大的情緒載體，是第九藝術。成立這間工作室，是因為我有一個跨越極端的創作野心：我們不只要做讓人『不敢動』的硬核恐怖遊戲，也要打造能讓人『感動落淚』的深刻神作。

為了實現這個願景，我目前正在集結一群帶有瘋狂靈魂與硬底子技術的創作者。團隊正經歷著高強度的實戰淬鍊，我們堅持極致的商業標準與開發紀律，因為唯有如此，才能將我腦海中構築的世界，透過團隊的雙手完美具象化，並震撼地遞交到玩家面前。

我們要向世界證明，何謂頂級的藝術體驗。

開發資金保障

本工作室已獲得天使投資人前期注資，具備穩健的開發資金底盤。

所有核心開發軟硬體、高階付費素材皆由工作室全力支援，讓團隊能心無旁騖地專注於品質。

擁有明確的階段性資源挹注，確保我們的每一個瘋狂點子，都能無後顧之憂地推進至 Steam 殿堂。

Ceilusion Roadmap:我們的願景

📍 Phase 1:【菁英篩選與預備期】 (即日起 ~ 2027.02.04)
「這是一張通往商業大作的入場券，但你必須證明你拿得動。」

此階段為團隊的預備核心期。我們將透過多次高強度的 Game Jam 進行實戰壓力測試。這是一場汰弱留強的生存戰，唯有在技術與心態上展現出絕對成長的開發者，才能正式取得核心成員的席位與未來分潤權。

📍 Phase 2:【管線驗證與首戰】 (2027.02.04 ~ 2027.05.04)
「不紙上談兵，用市場驗證我們的刀鋒。」

倖存的預備核心將正式簽署《核心成員最高開發準則》，並成為核心成員。我們將用 3 個月的時間，啟動並完成第一款 Steam 小品遊戲的開發與上架。這個階段的目標不是賺大錢，而是徹底跑順團隊的商業開發Workflow。

📍 Phase 3:【商業級標準打磨】 (2027.05.04 ~ 2027.07.01)
「從『會做遊戲』到『能做爆款』的跨越。」

針對首款小品遊戲的市場反饋與團隊協作進行覆盤。在這兩個月內，我們將進行高強度的持續磨合，確保團隊的技術力、美術質感與執行效率，完全對齊國際「商業爆款」的絕對標準。

📍 Phase 4:【旗艦恐怖大作啟動】 (2027.07.01 ~ 2029.07.01)
「磨刀結束，正片開始。」

團隊戰力成型，正式啟動 Ceilusion 第一款「商業級恐怖遊戲」的全面開發。我們將挹注所有的資金、技術與行銷資源，打造一款能在 Steam 市場上引起轟動的指標性作品。

📍 Phase 5:【IP 宇宙擴張】 (2029.07.01 ~ 2031.07.01)
「用爆款複製爆款，放大商業利潤。」

延續第一款恐怖遊戲的成功與聲量，直接啟動該 IP 的第二款續作研發。透過已成熟的開發管線與粉絲基數，實現獲利的指數型增長。

🚩 Phase 6:【財富自由與傳奇落幕】 (2032)
「我們來這裡不是為了當一輩子社畜，是為了翻轉人生。」

團隊核心成員透過遊戲分潤達成「財富自由」之目標。Ceilusion 將正式宣告解散。我們不追求成為百年企業，我們只做遊戲圈最傳奇的開發部隊。任務完成，光榮退役。

團隊分工與戰力藍圖

以下為 Ceilusion 為了實現商業級開發，所構築的核心成員組織架構。這裡明定了各位置的完整職責與絕對的交付標準；此架構將隨專案推進與戰略需求持續進化。

你為何加入

1. 股東級的財富分配

我們不是在找免費勞力，我們是在尋找未來的「商業合夥人」。

  • 遊戲在 Steam 上架後的每一筆營收（扣除成本），都將依據核心成員的貢獻權重進行分潤。
  • 我們不發死薪水，因為我們的目標是讓戰友達成財富自由。榮耀與利潤，我們共享。

2. 絕對的商業實戰武裝

資本與市場從不看你的潛力，只看你「成功發行過什麼產品」。

在這裡，你將經歷業界最高標準的開發 Workflow。這不是練習，而是一場真槍實彈的市場肉搏。當專案落地，你擁有的將是業界最硬的通行證——「一款真正賺錢的商業大作」。

如何加入我們？

目前 Ceilusion 第一階段的開發編制已經滿員，但我們的大門永遠為充滿熱情的創作者敞開。

我們深知，未來的頂級戰力往往來自於對遊戲純粹的渴望與不斷成長的潛力。你不一定要是滿級的大神，只要你對我們的願景有共鳴，且有想跟我們一起做點酷東西的野心，都歡迎把你的作品集丟過來。
